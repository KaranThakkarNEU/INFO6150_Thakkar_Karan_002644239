**Overview**

Static portfolio website developed using HTML5 & CSS3.

**Tags used for this Assignment are as follows with description:**

- Favicon
  - Use for displaying an icon on the window tab of the website
- Nav
  - To create navbar(semantic tag)
- Ul
  - list of item inside menu tag for top menu list
- Section
  - to create separate sections for certificates, home, resume, contact
- Input, label
  - to create form fields
- p
  - for displaying text
- Link
  - to link external css file
- Form
  - contact me form
- Images
  - To display certificates
- Hyperlink
  - to navigate different sections
- Button
  - resume download button
- header
  - Top navbar to display menu items and name
- footer
  - displays copyright details
- menu
  - use to display menu items
- tel for contact information
  - in about section to display phone number
- mailto for contact information
  - in about section to display personal email id
- figure
  - to display certificates with figcaption tag to display text
- hover
  - on resume button(changes color of button)
- media queries
  - to make website responsive
- absolute & relative properties
  - to style text and navbar menu items
- iframe
  - to display personal introduction video in about section
- transitions
  - to transition menu open & close on mobile devices
- pdf download option
