**Overview**

Static website developed using HTML5 & CSS. Website is about renting cars on an hourly basis, has some informational videos on how to rent cars, has an faq section, and a form to contact customer care.
 
**Tags used for this Assignment are as follows with description:**

* Favicon
  * Use for displaying an icon on the window tab of the website
* Table
  * Use to show a comparison table between two cars
* Nav
  * To create navbar(semantic tag)
* Ul
  * list of item inside menu tag for top menu list
* Section
  * to create separate sections for videos, faq, rental car
* H2, H4
  *  For heading in different sections
* IFrame
  * to show YouTube videos on website
* Input, label
  * to create form fields
* p
  * for displaying text
* Link
  * to link external css file
* Form
  * contact us form, rent car form, sell car form(all are one)
* Images
  * To display available rental car cards
* Hyperlink
  * to navigate to the rent car contact form from the home page and navigate back to the home page
* Button
  * On the rent car card and also for the login button
* audio
  * Inside FAQ to read and hear the answer
* video
  * In the video section for information on how to rent a car
* header
  * Top navbar to display menu items and company logo
* footer
  * Displays company description and contact information
* summary
  * Used in the FAQ section
* menu
  * use to display menu items
* tel for contact information
  * in the footer section to display the telephone number of the Company
* mailto for contact information
  * in the footer section to display email address of Company
